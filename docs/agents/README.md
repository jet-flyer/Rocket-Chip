# docs/agents/

Operational discipline files for AI agents working in this repository.

These were previously under `.claude/` and have been relocated here so they are accessible to all agents (not Claude-specific).

## Contents

- `SESSION_CHECKLIST.md` — Session hygiene rules, commit/push/end procedures, and trigger-driven doc edits.
- `PROTECTED_FILES.md` — Files requiring explicit user permission before editing.
- `LESSONS_LEARNED.md` — Debugging journal and project knowledge base.
- `DEBUG_PROBE_NOTES.md` — OpenOCD, GDB, and hardware probing notes.
